## 필수 라이브러리 설치
다음 명령어를 실행하여 필요한 종속성을 설치합니다:
```bash
pip install fastapi transformers torch uvicorn
```

## 서버 실행
```bash
python api.py
```

## API에 접속
애플리케이션이 실행되면 다음에서 API에 접속할 수 있습니다:
```bash
http://127.0.0.1:8000
```
